## 一、决策逻辑链（如果…那么…）

### 第一层：判断「现在处于哪个阶段」

这是所有操作的前提，先定阶段再做其他判断。

**情绪七阶段判断链**

> 原话逻辑路径：运辖期 → 发酵期 → 加速期 → 分歧期 → 高潮期 → 扩散期 → 退潮期

| 观察到的现象 | 判定为 | 对应操作 |
|------------|--------|---------|
| 板块有涨停但不持续，市场不跟随 | 运辖期 | 观察，不做或极轻仓 |
| 板块开始有二板出现，周边板块开始联动 | 发酵期 | 开始建仓，选主龙头 |
| 二板数量快速增加，量能放大，资金聚集 | 加速期 | 重仓持有，买低位票，不买高位 |
| 龙头出现分歧（炸板/高开低走），但还有二三板 | 分歧期 | 选跟随市场走势的票，不选独立走势的票 |
| 二板、三板数量比加速期更多，市场全面亢奋 | 高潮期 | 最高仓位，快速博弈 |
| 板块开始挖掘新标的（属性外延）、资金向外扩散 | 扩散期 | **危险信号，开始减仓** |
| 板块不再产生新的高板，资金撤退 | 退潮期 | 减至3成以下，转低位补涨逻辑 |

**情绪路径有三种，不是每次都走完**

- 如果情绪走完整路径（运辖→退潮），那么可以按节点操作
- 如果情绪在分歧期后直接收缩（不出高潮期），那么没有高潮期这步，不能等高潮
- 如果情绪直接爆发后快速退潮，那么极短极快，要快进快出
- 如果退潮后出现地天板（低开涨停），那么成功率已降低，仓位要比第一次分歧期时更小

**一个周期多长时间**

- 如果是正常节奏，那么一个完整情绪周期大约 7-12 天
- 如果进入修复期，那么随机性极强，要降仓、等待下一个周期重启

### 第二层：判断「这只票值不值得跟」

**领头羊识别链**

- 如果一只票是真正领头羊，那么它能带动整个板块跟随，涨它的时候周边也在涨
- 如果出现竞争/卡位（同板块两只票争龙头），那么资金分散，两只都做不好
- 如果跟错了领头羊（中毒现象），那么选的人多但涨幅反而不如另一只，要换
- 如果用凯恩斯选美原则，那么选的不是自己认为最好的，而是判断大多数人会选哪只

**位阶判断链**

- 如果一只票是首板（第一次涨停），那么用来定焦题材，判断板块是否值得参与
- 如果是二板（连续第二个涨停），那么用来确认强度，重点观察是缩量一字板还是秒封
- 如果二板是缩量一字或秒板封停，那么说明筹码高度锁定，是强信号
- 如果二板是经过打开换手才封，那么筹码置换充分，同样可以参与但风险稍高

**扎板修复链（运辖期特有）**

- 如果一只票在运辖期出现扎板（涨停打开），那么不要马上放弃，观察能否被修复
- 如果扎板幅度大但三天内被修复（价格重新突破扎板高点），那么是强信号，可以在突破点或回档时买
- 如果扎板之后出现跳空低开直接大阴线，那么不是修复，是死亡形态，直接放弃
- 如果运辖期扎板修复成功，那么这个模式叫「扎板越点谷」，成功后大概率会有人气爆发

**二波龙头时间窗口**

- 如果某只票是前一波的龙头，那么它有可能成为二波龙头
- 如果二波龙头没有在 5-7 天内重新启动，那么时间窗口已过，不再是二波龙头候选

---

### 第三层：判断「什么时候进、什么时候出」

**竞价阶段判断链**

- 如果竞价高开超过预期且竞价量大于昨日总量的 10%，那么判定为超预期，可以参与
- 如果 9:20 之后单子已不能撤，那么竞价阶段的入场价就是锁定价，要在 9:20 前确认
- 如果竞价阶段开盘换手率高且开盘金额排名靠前，那么这只票是当天优先选择

**盘中分时判断链**

- 如果分时第二波回落幅度不超过第一波的一半，那么多头结构完整，可以继续持仓
- 如果分时第二波回落超过第一波的一半，那么结构破坏，减仓或止损
- 如果分时斜率越来越陡（每一段上涨的角度在加大），那么说明资金在持续加速流入
- 如果分时出现托单或压单，那么说明有主力在控盘，行为逻辑不同于自然成交，要单独判断

**量价关系判断链（三种市场环境）**

- 如果是熊市中出现放量，那么方向向下（看空），放量是坏事
- 如果是牛市回调时出现放量，那么是新的一波段即将开始（看多），放量是好事
- 如果是震荡市，那么量价关系参考意义下降，看情绪周期更重要

**卖点判断链**

- 如果扩散期板块连续两天没有挖掘新的题材标的，那么赚钱效应在收缩，准备撤退
- 如果两个主力板块出现互相拆台（A涨B跌），那么市场流动性不足，赚钱效应差
- 如果两个板块共振拉升，那么是赚钱效应良好的信号，可以加仓
- 如果买入后第二天开盘十分钟内出现一字或快速拉升，那么持续性强，可以加仓

**加速期的特殊买法**

- 如果处于加速期，那么买低位票，不买高位票
- 如果是一字板启动（开盘直接封板），那么不选它（买不到，也缺乏分歧空间）
- 如果加速转分歧，那么在分歧期选「跟随市场情绪走」的票，不选「独立走势」的票

### 第四层：判断「押多少」（石头老师 B 的内容）

**期望值门槛（所有下注的前提）**

- 如果一笔交易的期望值（赔率 × 胜率）小于 1，那么这笔交易不值得做，不管感觉多好
- 如果连续参与负期望的交易，那么长期一定亏损，无论短期是否赚钱

**凯利公式（最佳仓位数学）**

```
F（最佳仓位比例）= （赔率 × 胜率 - 败率）/ 赔率
```

- 如果按凯利公式算出最佳仓位 F 下注，那么长期来看复利增长最快
- 如果用满仓（100%），那么只需要一次全输就归零，必死
- 如果每次都用最佳仓位的一半（F÷2）下注，那么收益更稳健，回撤更小，实战更推荐
- 如果人脑对小概率事件有高估偏差，那么会在不值得的地方过度下注，要用数学纠正直觉

**仓位与周期的联动**

- 如果处于情绪启动期（首板/二板刚出现），那么仓位开始建仓（3-5成）
- 如果处于主升期，那么重仓（5-8成）
- 如果处于高位震荡/衰退期，那么由8成降到3成以下
- 如果处于修复期/冰点，那么空仓或极轻仓（低于1成）

## 五、全天两位主讲人的核心立场对比

|  | **主讲人 A** | **石头老师 B** |
|--|------------|--------------|
| 核心方法论 | 情绪周期 × 领头羊借势 | 期望值 × 凯利公式 |
| 选股逻辑 | 凯恩斯选美（选大多数人会选的） | 不看选哪只，看期望值是否为正 |
| 仓位方法 | 情绪阶段决定仓位比例（7级联动） | 数学公式算出最佳仓位（F/2原则） |
| 止损方式 | 扩散期信号出现即减仓 | 封杀亏损（严格止损10%） |
| 底层哲学 | 市场是人心的博弈，不是价格的博弈 | 市场是概率游戏，用数学管理随机性 |

> 两套体系并不冲突：A 解决「什么时候做、做哪只」，B 解决「做多少、亏多少就止损」
